Viable Continuation Capacity

Tiantian Zhang · Department of Computer Science, Columbia University, A local analogue of epiplexity for counterfactual internal updates

Question

Epiplexity asks what structural information is accessible to a computationally bounded learner. Can we define a local, state-conditioned analogue: given an internal state M, how many distinct coherent continuation modes does it make reachable under finite budget?

1. Connection to Epiplexity

The Epiplexity paper moves beyond raw Shannon/Kolmogorov information toward structural information accessible to bounded learners. This note asks a local version of that question: not “how much structure is in a dataset?”, but “how much coherent continuation structure is supported by this internal state?”

Epiplexity

Structural information in data for a bounded learner.

Viable continuation capacity

Coherent continuation modes reachable from an internal state under finite rollout.

2. Definition

Symbol Meaning
M Current internal state, e.g. a hidden state of a language model.
B Finite compute / sampling / token budget.
T Finite rollout horizon.
γ A sampled continuation path from M.
V(γ) Viability verifier: whether the continuation is coherent / valid for the task.
ψ(γ) Semantic mode map: the continuation’s outcome type among viable paths.

Define viable path entropy, or mirror horizon, by:

H_{B,T}(M) = log Pr[V(γ)=1] + H( ψ(γ) | V(γ)=1 )

This is high when two things are true:

  1. continuations are likely to remain viable;
  2. among viable continuations, many distinct semantic modes remain reachable.

This intentionally differs from raw entropy: random branching does not count unless it survives the viability filter.

3. Candidate update

Let p be a candidate update, prompt edit, steering intervention, retrieved document, or concept redefinition. Suppose integrating it induces a counterfactual internal state M^(p).

Define the update’s value as horizon gain:

CC_{B,T}(M,p) = H_{B,T}(M^(p)) - H_{B,T}(M)

Informally:

A useful update is one that increases viable continuation capacity.

It should not merely be similar, novel, high-probability, high-entropy, or far in hidden-state space. It should open more coherent structured continuations.

4. Concrete example: concept redefinition

Consider an in-context redefinition:

In this context, "dog" means cat.

A good update should:

  • make dog behave like cat inside the redefinition context;
  • preserve ordinary dog / cat behavior outside that context;
  • preserve unrelated concepts and grammatical coherence.

Compare:

Condition Intervention Expected issue / benefit
Prompt-only No hidden-state intervention May partially redefine the concept but inconsistently.
Ungated steering h' = h + s Can move the target concept, but may damage ordinary contexts.
Gated steering h' = h + sigmoid(h^T v)s Should apply the shift selectively when the redefinition is active.

Prediction

At matched target effect, gated steering should increase H_{B,T} more than ungated steering, because it should preserve more viable continuation modes with less collateral damage.

5. Concrete feedback question

Is viable continuation capacity a meaningful local analogue of epiplexity, or does the epiplexity framework already suggest a cleaner formulation for this object?

More specifically:

  • Is the viability-filtered entropy form well-motivated?
  • Should ψ be learned, hand-specified, or derived from semantic representation geometry?
  • Is concept redefinition a reasonable minimal experiment for estimating this quantity?
