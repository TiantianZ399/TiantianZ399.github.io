Compatibility-Preserving Steering for In-Context Concept Redefinition

Tiantian Zhang ·Department of Computer Science, Columbia University A short project explainer for an NLP/interpretability audience. It presents the experiment in concrete terms and keeps the heavier theory language in the background.

One-sentence version

Can we steer a language model toward a contextually redefined concept only when that redefinition is active, while preserving ordinary behavior elsewhere?

Motivation Vocabulary translation Toy task Method Metrics Experiment Fit with Hewitt lab Short email

1. Motivation

In-context learning can redefine concepts. For example, a prompt can say: 

In this game, the word "dog" means cat.
Question: A dog says meow. Is that expected?

A model may partially treat dog as cat inside this context. This is useful for studying concept binding, model editing, steering, and controllability.

But there is a subtle problem:

A good intervention should not simply push the representation of dog toward cat everywhere. It should make that shift only in contexts where the redefinition is active.

So the central question is not only can we steer? It is:

Can we steer selectively, achieving the target change while minimizing collateral damage?

2. Vocabulary translation

The broader theory uses terms like “underdetermination” and “constructive compatibility.” For a first conversation with an NLP researcher, it is clearer to foreground the experimental version:

Theory term NLP-facing phrase Plain meaning
underdetermination underspecified context / ambiguous context The context does not uniquely determine what the model should internally do next.
constructive compatibility compatibility-preserving steering Move the target concept in the desired direction without disrupting unrelated behavior.
rupture collateral damage The intervention damages non-target concepts, ordinary contexts, grammar, or control tasks.
invariant survival preserving non-target behavior Important behaviors stay stable after the intervention.
mirror hidden state / internal representation The model’s current internal representation at a layer and token position.
possibility candidate concept update A possible re-interpretation, continuation, or steering target.

Recommended framing: “selective steering for concept redefinition with low collateral damage.”

This wording puts the empirical question first: when to steer, what to preserve, and how to measure the tradeoff.

The denser theory phrase “underdetermined mirror evolution by constructive compatibility” can be saved for a theory appendix or follow-up discussion.

3. Toy task: in-context concept redefinition

The toy setting is intentionally simple because it lets us measure success and side effects cleanly.

Redefinition Target effect Non-target behavior to preserve
"dog" := "cat" Inside the redefinition context, dog behaves more like cat. Outside the context, ordinary dog and cat behavior remains intact.
"Paris" := "London" Inside the context, questions about Paris follow London-like facts. Ordinary geography facts outside the context are preserved.
"red" := "blue" Inside the context, red maps to blue-like behavior. Other color words and ordinary color knowledge are preserved.

In this toy task, “underspecified” means the context creates two plausible interpretations: the original lexical meaning and the temporary in-context meaning.

4. Method: gated steering

Standard steering applies the same vector everywhere:

\[ h' = h + s \]

But concept redefinition is context-specific. The intervention should fire only when the current hidden state indicates that the redefinition is active. So use a gate:

\[ h' = h + \sigma(h^\top v)s \]

Interpretation:

  • \(s\) learns what change to make.
  • \(v\) learns when to apply that change.
  • \(\sigma(h^\top v)\) is the selectivity gate.

Core hypothesis

Gated steering should achieve similar target movement as ungated steering, but with lower collateral damage on non-target contexts.

5. Metrics

5.1 Target effect

Does the redefined concept move toward the intended behavior?

\[ \text{TargetEffect} = \text{Score}_{\text{target}}(h') - \text{Score}_{\text{target}}(h) \]

Example: after "dog" := "cat", does the model answer more cat-like questions for dog inside the redefinition context?

5.2 Collateral damage

Does the intervention harm unrelated behavior?

\[ \text{Damage} = \sum_i w_i \max\{0, g_i(h)-g_i(h')\} \]

Here \(g_i\) are probes or behavioral tests for non-target properties: ordinary word meaning, grammar, factual consistency, unrelated concepts, and control tasks.

5.3 Selectivity

Does the gate fire only in redefinition contexts?

\[ \text{Selectivity} = \mathbb E[g(h)\mid \text{redefinition active}] - \mathbb E[g(h)\mid \text{ordinary context}] \]

where \(g(h)=\sigma(h^\top v)\).

5.4 Tradeoff score

The practical objective is:

\[ \text{UsefulSteering} = \text{TargetEffect} - \lambda \text{Damage} - \mu \text{InterventionSize} \]

This is the NLP-facing version of “constructive compatibility.”

6. Minimal experiment

  1. Create redefinition prompts.
    Examples: dog := cat, Paris := London, red := blue.
  2. Collect hidden states.
    Record residual-stream states at selected layers and positions.
  3. Train or estimate concept directions.
    Learn a target steering vector \(s\) for the redefinition.
  4. Train a gate.
    Learn \(v\) so the intervention fires in redefinition contexts but not ordinary contexts.
  5. Compare three conditions.
    1. Prompt-only redefinition.
    2. Ungated steering: \(h'=h+s\).
    3. Gated steering: \(h'=h+\sigma(h^\top v)s\).
  6. Evaluate target effect, collateral damage, and selectivity.

Expected result

Gated steering should sit on a better target-effect / collateral-damage frontier: similar target movement, less damage.

7. Baselines and controls

Concern Control
The probe is doing all the work. Use control tasks and compare against random labels / lexical controls.
The intervention is just lexical replacement. Condition on token identity; test whether the vector decomposes into a few token embeddings.
Ungated steering is already enough. Compare target effect at matched collateral damage.
The gate fires everywhere. Measure gate values on ordinary contexts and unrelated redefinition contexts.
The result is concept-specific. Run several redefinition pairs across animals, places, colors, professions, and abstract labels.

8. Why this could fit Prof. Hewitt’s lab

This project naturally connects to several themes in Prof. Hewitt’s work:

  • Probing and representation geometry. Structural probes ask whether linguistic structure is encoded in representation geometry. Structural Probe paper.
  • Probe controls. The project needs careful controls so target-effect measurements are not just probe artifacts. Control Tasks paper.
  • Usable information beyond baselines. The project asks whether the gated intervention adds usable context-specific control beyond lexical identity and prompt-only behavior. Conditional Probing paper.
  • Precise control of neural language systems. The project is directly about steering only when appropriate, not globally. John Hewitt homepage.

9. Scope boundaries

  • This is a narrow, testable intervention study, not a broad theory-of-everything claim.
  • The goal is not just to probe for a concept, but to test whether a gate improves context-specific steering.
  • The intervention is local and context-conditioned, rather than a form of global model editing.
  • The value should come from clear success and failure criteria, not from terminology alone.

The specific empirical claim is small:

For in-context concept redefinition, gated steering can improve the target-effect / collateral-damage frontier.
